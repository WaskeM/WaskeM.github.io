About the project

The project is intended for the online sale of books on the Internet. The complete project is written in the Java programming language. The project consists of two parts that work together.

The first part is backend, for the admin, some of the functionalities are: The second part is the customer interface, some of the functionalities are:
  • User management (including admin login / logout),
  • Category management,
  • Book management,
  • Management overview,
  • Customer management,
  • Order management,
  • Statistics (admin panel).
  • Home page: list of recently published books; best-selling books; most favorite books,
  • List of books in category,
  • View book details,
  • Search for books,
  • Client registration,
  • Write book reviews,
  • Place an order
